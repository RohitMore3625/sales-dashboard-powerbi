# 📊 Sales Overview Dashboard (Power BI)

## 🔍 Overview

This Power BI dashboard provides a comprehensive analysis of sales performance across regions, products, customers, and time. It enables data-driven decision-making through interactive visualizations and KPIs.

---

## 🎯 Key Features

* 📌 KPI Cards: Revenue, Profit, Quantity, Transactions, and COGS
* 📈 Revenue vs Target trend analysis (monthly)
* 🌍 Country-wise Revenue & Profit comparison
* 🏆 Top 5 Products by Units Sold
* 👥 Top Customers by Revenue
* 🌎 Region-wise Revenue Distribution
* 🎛️ Interactive Filters (Year, Category, Customer, Product)

---

## 💡 Key Insights

* Revenue shows consistent performance with peaks around mid-year
* France and USA are top revenue-generating countries
* Certain products (e.g., Galaxy S23, HP Spectre) dominate unit sales
* Customer segmentation highlights key revenue contributors
